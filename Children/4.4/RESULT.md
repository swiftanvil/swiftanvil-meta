## Key Design Decisions

- **Read-only by design**: No installation, configuration, or privileged changes. Pure detection.
- **Safe auth probes**: Only `--version` and local config checks. No network calls.
- **Optional agents warn, not fail**: Missing Claude/Codex/Gemini is a warning, not a failure.
- **Exit codes**: 0 for pass/warn, 1 for any fail — CI-friendly.
- **Actor isolation**: `CapabilityDiscovery` is an actor for safe concurrent access.
